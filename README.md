# QuantumTouch

A mesmerizing interactive particle system that responds to your hand gestures in real-time. Built with Three.js and MediaPipe Hand Tracking.

[**🔴 Live Demo**](https://vinitsingh665.github.io/QuantumTouch/)

## ✨ Features

- **Gesture Control**: Use your hands to select, rotate, and zoom into shapes.
- **Real-time Physics**: Thousands of particles reacting smoothly to interactions.
- **Multiple Visual Sets**: Explore diverse themes like Cosmic, Mystical, Retro, Marine, and more.
- **Web-Based**: Runs entirely in the browser using WebGL and Webcam input.

## 🎮 How to Play

Ensure your camera is enabled and your hands are visible.

### 1. Shape Selection
- **Single Hand**: Show **1 to 5 fingers** to switch between shapes in the current set.
- **Dual Hand**: Show the **SAME number of fingers** on **BOTH hands** to confirm a selection.

### 2. Manual Rotation
- Make a **FIST ✊** with **ONE hand**.
- Use your **OTHER hand ✋** to grab and rotate the object in 3D space.

### 3. Zoom Control
- Use **BOTH hands** to select a shape.
- Move hands **APART** to **Zoom Out**.
- Move hands **TOGETHER** to **Zoom In**.

## 🛠️ Technologies Used

- **Three.js**: For high-performance 3D particle rendering.
- **MediaPipe Hands**: For precise real-time hand tracking and gesture recognition.
- **HTML5 & CSS3**: For the UI interface.

## 🚀 Setup

Simply open `index.html` in a modern web browser, or host it on any static server.
